Curiosidade do dia: HTML e CSS, por muitos, não são consideradas como linguagens de programação pelo simples fato de não ser possível criar funções ou rotinas para executarem tarefas.

HTML é considerada uma linguagem de marcação, como sua própria sigla nos diz "Linguagem de marcação de Hipertexto".

CSS é uma linguagem de estilização que possui uma única finalidade, configurar todo o estilo de uma página web criada com a linguagem de marcação HTML.
